# 🖼️ How to Add Gallery Images Using URLs

## ✅ I Just Fixed This For You!

You can now **paste image URLs directly** into the gallery section!

---

## 📝 Step-by-Step Instructions

### 1. Get Image URLs from Zoopla

For each gallery image on Zoopla:
1. **Right-click the image**
2. Select **"Copy image address"** or **"Copy image link"**
3. The URL is now in your clipboard (looks like: `https://lid.zoocdn.com/...`)

### 2. Add to Your Admin

After you've filled in the main details:

1. Scroll down to the **"Or paste image/video URLs"** section
2. **Paste the URL** you copied
3. **Press Enter**
4. The image will appear in the gallery list below
5. Repeat for each gallery image

---

## 🎯 Complete Workflow for One Property

### Step 1: Copy Main Image
```
1. On Zoopla: Right-click main image → Copy image address
2. In Admin: Paste into "Main Image URL" field at the top
```

### Step 2: Copy Gallery Images
```
1. On Zoopla: Right-click 2nd image → Copy image address
2. In Admin: Scroll to "Or paste image/video URLs"
3. Paste URL → Press Enter
4. Repeat for images 3, 4, 5, etc.
```

### Step 3: Fill Other Details
```
- Title
- Location
- Price (number only, no £)
- Beds, Baths, SqFt
- Description
```

### Step 4: Save
```
Click "Create Listing"
```

## ⚡ Quick Tips

**Reorder Images:**
- Use the ↑ ↓ arrows to change order
- First image in gallery shows first on the property page

**Set as Main:**
- Click the 🖼️ icon to make any gallery image the main image

**Remove Image:**
- Click the ✕ (trash) icon to remove

**Add Multiple:**
- Paste URL → Press Enter
- Paste URL → Press Enter

- As many as you want!

## 📋 Example

**From Zoopla:**
```
Main: https://lid.zoocdn.com/645/430/main.jpg
Gallery 1: https://lid.zoocdn.com/645/430/img1.jpg
Gallery 2: https://lid.zoocdn.com/645/430/img2.jpg
Gallery 3: https://lid.zoocdn.com/645/430/img3.jpg
```

**In Admin:**
1. Paste main URL in "Main Image URL" field
2. Paste img1.jpg in "Or paste image/video URLs" → Enter
3. Paste img2.jpg in "Or paste image/video URLs" → Enter
4. Paste img3.jpg in "Or paste image/video URLs" → Enter
5. Done! ✅

## ❓ FAQ

**Q: Do I need to upload images?**
A: No! Just paste the URLs.

**Q: Will Zoopla images work?**
A: Yes! They're hosted on Zoopla's CDN.

**Q: How many gallery images can I add?**
A: As many as you want!

**Q: Can I mix uploaded and URL images?**
A: Yes! Use both methods if you want.

**Q: What if I paste a wrong URL?**
A: Just click the ✕ icon to remove it.
